# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-28

### Added

#### Core Protocols
- X402PaymentProcessor - Multi-token payment processor
- XULSwap Factory - AMM DEX factory
- XULSwap Router - DEX router for token swaps
- WXUL - Wrapped XUL token
- WUSDT, WUSDC, WBTC, WETH - Wrapped cross-chain tokens

#### Advanced Features
- XULSmartWallet - Smart contract wallet with batch transactions
- XULAIAgentRegistry - AI agent registration (ERC-8004)
  - AI identity NFTs
  - Behavior recording
  - Reputation system
- XULzkMLVerifier - Zero-knowledge ML verification
  - Model registration
  - ZK proof verification
  - Verifier management
- XULDePIN - Distributed computing network
  - Device registration (GPU/CPU/IoT/Storage/Edge)
  - AI task distribution
  - Reward mechanism
- XULTranslator - AI-powered translation system
  - 15 language support
  - AI + human translation
  - Quality verification
  - Translator reputation

#### Infrastructure
- Hardhat configuration for XUL Chain
- Deployment scripts
- Test suites
- TypeChain type bindings
